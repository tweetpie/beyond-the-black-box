# Beyond the Black Box Dataset

This dataset was utilized in the paper **"Beyond the Black Box: Programmable AI and Explainable Text Analysis for Trustworthy Social Intelligence."** 

The dataset contains Tweet IDs corresponding to tweets collected from Twitter (X) related to global energy trends, geopolitical discourse, and energy-sector companies.

To comply with Twitter’s Terms of Service, we only release the **Tweet IDs** of the collected tweets. Researchers can use the Tweet IDs to rehydrate the tweets using the official Twitter API.

If you use this dataset in your research, please cite the corresponding paper.
